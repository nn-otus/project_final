# Команды запуска виртуальных сетей

```bash
sudo virsh net-start front-net
sudo virsh net-start back-net
sudo virsh net-start db-net
```

# Включение автозагрузки

```bash
sudo virsh net-autostart front-net
sudo virsh net-autostart back-net
sudo virsh net-autostart db-net
```

# Как проверить статус

```bash
sudo virsh net-list --all
```
В колонке State должно быть active, а в Autostart — yes
Запуск машин выполнять только после активизации всех сетей
